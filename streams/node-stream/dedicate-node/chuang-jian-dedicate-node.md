### 配置Dedicate Node

<figure><img src="../../../.gitbook/assets/image (69).png" alt="" width="375"><figcaption></figcaption></figure>

* 節點類型：支持BSC全節點，Solana節點即將上線，敬請期待
* 區域：支持法蘭克福和弗吉尼亞2個可用區。BlockRazor會動態監測可用區資源，如資源不足則無法創建Dedicate Node，請和我們取得[聯繫](https://discord.com/invite/qqJuwRb8Nh)
* 服務器配置：根據可用區顯示相應的Dedicate Node服務器配置
* 節點客戶端：支持Geth客戶端
{% endstep %}

{% step %}
### 選擇服务週期和支付方式

<figure><img src="../../../.gitbook/assets/Snipaste_2025-05-15_14-37-58.png" alt="" width="375"><figcaption></figcaption></figure>

目前僅支持加密貨幣支付，支持的加密貨幣為USDT.BEP20、USDT.ERC20和USDT.PRC20
{% endstep %}
